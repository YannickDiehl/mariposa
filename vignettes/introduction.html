Overview

SurveyStat is designed to provide comprehensive tools for statistical analysis of survey data. This vignette introduces the basic functionality of the package.

Basic Usage

Descriptive Statistics

The package provides comprehensive descriptive statistics for survey data:

# Load the synthetic survey data
data(survey_data)

# Basic descriptive statistics
survey_data %>% describe(age, income, weights = sampling_weight)
#> 
#> Weighted Descriptive Statistics
#> ------------------------------- 
#>  Variable     Mean Median       SD Range  IQR Skewness Effective_N
#>       age   50.514     50   17.084    77   25    0.159      2468.8
#>    income 3743.099   3500 1423.966  7200 1900    0.724      2158.9
#> ------------------------------------------------------------------

# Weighted mean calculation
survey_data %>% w_mean(life_satisfaction, weights = sampling_weight)
#> 
#> Weighted Mean Statistics
#> ------------------------ 
#> 
#> --- life_satisfaction ---
#>           Variable weighted_mean Effective_N
#>  life_satisfaction         3.625      2390.9

Statistical Tests

Perform hypothesis tests with survey data:

# Group comparison with t-test (male vs female)
survey_data %>% 
  t_test(life_satisfaction, group = gender, weights = sampling_weight)

Package Features

SurveyStat provides a comprehensive suite of statistical tools:

  • Descriptive Statistics: describe(), frequency(), weighted w_* functions
  • Hypothesis Testing: t-tests, ANOVA, non-parametric tests
  • Post-Hoc Analysis: Tukey tests, estimated marginal means
  • Survey-Weighted Analysis: Proper handling of sampling weights
  • Synthetic Datasets: License-free data for testing and examples
