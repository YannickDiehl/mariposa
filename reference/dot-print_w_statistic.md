# Generic print method for w\_\* statistic objects

Shared print implementation used by all standard w\_\* functions.

## Usage

``` r
.print_w_statistic(x, stat_label, weighted_col, unweighted_col, digits = 3)
```

## Arguments

- x:

  A w\_\* object

- stat_label:

  Display name (e.g., "Mean", "Standard Deviation")

- weighted_col:

  Column name for weighted values

- unweighted_col:

  Column name for unweighted values

- digits:

  Number of decimal places
