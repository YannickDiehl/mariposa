# Extract codebook metadata for a single variable

Extract codebook metadata for a single variable

## Usage

``` r
.extract_var_metadata(x, var_name, position, max_values = 10)
```

## Arguments

- x:

  Data vector

- var_name:

  Variable name

- position:

  Column position in original data

- max_values:

  Maximum number of values to display before truncating

## Value

Named list of metadata
