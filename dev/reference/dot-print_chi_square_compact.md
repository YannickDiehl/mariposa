# Print a compact one-line summary for a single chi-squared test

Print a compact one-line summary for a single chi-squared test

## Usage

``` r
.print_chi_square_compact(results, i, var_label, weighted_tag, digits)
```

## Arguments

- results:

  Data frame with chi-squared results

- i:

  Row index

- var_label:

  Variable label string (e.g. "gender x region")

- weighted_tag:

  Weighted tag string (e.g. " \[Weighted\]" or "")

- digits:

  Number of decimal places
