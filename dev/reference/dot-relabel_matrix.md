# Relabel a matrix's dimnames using label maps

Relabel a matrix's dimnames using label maps

## Usage

``` r
.relabel_matrix(mat, label_maps, max_width = 20)
```

## Arguments

- mat:

  A matrix with dimnames

- label_maps:

  Named list of label maps (one per dimension)

- max_width:

  Maximum character width for truncation

## Value

Matrix with relabelled dimnames
